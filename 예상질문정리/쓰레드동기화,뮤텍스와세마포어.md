```
쓰레드간 공유한 데이터를 읽고 쓸 시에, 쓰레드간 실행순서에 따라 공유한 데이터를 읽고 쓰는 작업이 누락될 수 있으며,이로 인해 쓰레드의 비정상동작을 야기할 수 있습니다.쓰레드 동기화란 이러한 문제를 막기 위해 여러 쓰레드가 동시에 공유한 데이터를 읽고, 쓰지 못 하도록 하는 기술입니다.
```

